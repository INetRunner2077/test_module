Задание

 Нужно создать свой кастомный компонент с формой, состоящей из трех полей:
1. Имя
2. Фамилия
3. Имеет доступ (да\нет)
4. Номер телефона

Отправку данных с формы реализовать на языке javascript. Присланные ответы с формы нужно сохранять в свою орм таблицу. Так же, нужно написать миграцию для создания этой таблицы.

Как будет проходить проверка:
1. На чистый битрикс накатываются\устанавливаются ваши файлы.
2. Запускается миграция, либо происходит установка модуля.
3. Проверяется работа формы.
4. Проверяются записи в БД.


Выполнение 

1)	Добавьте прилагаемый архив в папку /bitrix/modules
2)	Установите модуль
3)	Добавьте на станицу компонент, он находиться в дереве “Свечка”-> “Компонент формы” (candle:candle)
4)	Модуль создаст ORM таблицу под названием “candle_form”
5)	При удалении модуль удалит созданную таблицу и компонент.
